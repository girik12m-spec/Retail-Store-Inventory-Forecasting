# Retail Store Inventory Forecasting

## Overview
This project analyzes retail inventory and predicts product demand using Machine Learning techniques.

## Features
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Training
- Prediction & Evaluation

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Results
- RMSE: 8.65
- MAE: 7.47

## Dataset
Retail inventory and sales dataset containing:
- Inventory levels
- Sales
- Orders
- Product categories
- Weather conditions
- Promotions
- Competitor pricing

## Project Structure
- Notebook for analysis and prediction
- CSV dataset
- Visualizations and insights
